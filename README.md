# 盛金敦煌风格 - Shengjin Dunhuang Style

融合中国传统艺术与西方华丽艺术的独特视觉风格。

## 风格定义

**盛金敦煌** = 盛唐雍容 + 金碧辉煌 + 敦煌飞天

核心特点：流光溢彩、极致华丽、金碧辉煌
